# Jeu Pong (Processing)

## Description
Jeu de type Pong réalisé en première année de Licence info.
Le joueur affronte un ordinateur en contrôlant une raquette.

## Fonctionnalités
- Déplacement avec les flèches
- Gestion des collisions
- Système de score
- Animation du jeu

## Technologies
- Processing (Java)
